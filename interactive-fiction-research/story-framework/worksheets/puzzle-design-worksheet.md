## Common Puzzle Pitfalls

**Avoid These:**

**Guess the Verb:**
- Problem: Player knows what to do but not which command to type
- Solution: Accept multiple synonyms, make command intuitive

**Pixel Hunting (Text Equivalent):**
- Problem: Required object is too obscure/hidden
- Solution: Make important objects findable through careful exploration

**Arbitrary Solutions:**
- Problem: Solution doesn't follow from clues
- Solution: Ensure every puzzle element is hinted at

**Moon Logic:**
- Problem: Solution only makes sense to creator
- Solution: Test with others, ensure logical progression

**Hidden Information:**
- Problem: Required info never provided
- Solution: Make all necessary information discoverable
