# Publisher Review Checklist

Use this checklist for systematic review at each milestone and final acceptance.

---

## PRE-REVIEW PREPARATION

### Materials Gathered
- [ ] Current deliverable received and accessible
- [ ] Original requirements/Publisher Brief available
- [ ] Previous milestone feedback available
- [ ] Quality standards documentation available
- [ ] Previous deliverables for comparison (if applicable)

### Review Context Set
- [ ] Understand what milestone this is
- [ ] Know specific milestone requirements
- [ ] Aware of any special considerations
- [ ] Time allocated for thorough review

---

## QUANTITATIVE CHECKS

### Deliverable Completeness
- [ ] All expected components received
- [ ] File format correct
- [ ] Naming convention followed
- [ ] Word count recorded

### Word Count Verification
- [ ] Total word count checked
- [ ] Within acceptable range for milestone
- [ ] Chapter count verified
- [ ] Chapter lengths reasonable

### Structural Completeness
- [ ] All required chapters/sections present
- [ ] Required scenes included
- [ ] Structural beats in correct positions
- [ ] Act proportions appropriate

---

## TECHNICAL QUALITY REVIEW

### Grammar & Mechanics
- [ ] Grammar scan completed
- [ ] Error density acceptable
- [ ] Style guide followed
- [ ] No systematic errors

### Spelling & Consistency
- [ ] Spelling checked
- [ ] Character names consistent
- [ ] Place names consistent
- [ ] Terminology consistent

### Formatting
- [ ] Document formatting correct
- [ ] Chapter formatting consistent
- [ ] Scene breaks properly marked
- [ ] Dialogue formatting correct

## CRAFT QUALITY REVIEW

### Prose Quality
- [ ] Prose reads smoothly

- [ ] POV maintained
- [ ] Tone matches requirements

### Readability
- [ ] Appropriate for target audience
- [ ] Clarity maintained
- [ ] Accessible vocabulary
- [ ] Engaging style

- [ ] Character arc progressing
- [ ] Supporting cast serving story
- [ ] Relationships developing

### Engagement
- [ ] Hook(s) working
- [ ] Reader investment maintained
- [ ] Emotional beats landing
- [ ] Desire to continue reading
