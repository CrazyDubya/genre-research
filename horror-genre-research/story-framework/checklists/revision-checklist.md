# Horror Story Revision Checklist

**Use this checklist to systematically revise your horror story.**
**Check each item as you complete it during revision.**

---

## Phase 1: Foundation Check

### Threat and Rules

- [ ] **Threat Clear**: Is the nature of the threat clearly established?
- [ ] **Rules Consistent**: Do you never break your own rules about how threat operates?
- [ ] **Rules Established Early**: Are threat's rules clear by 25% mark?
- [ ] **Threat Scary**: Is the threat genuinely frightening?
- [ ] **Threat's Weakness**: Is there a way to defeat or survive threat?

### Concept Clarity

- [ ] **Horror Type Clear**: Is it clear what type of horror this is?
- [ ] **Type Consistent**: Do you stick to one type (no mixing incompatible mechanics)?
- [ ] **Hook Strong**: Is the horror concept compelling and unique?
- [ ] **Fear Factor**: Does story tap into primal fear?

---

## Phase 2: Character Check

### Hero Character

- [ ] **Vulnerable**: Is hero genuinely vulnerable to threat?
- [ ] **Empathetic**: Does reader care about hero?
- [ ] **Active**: Does hero make choices (not just react)?
- [ ] **Growth Arc**: Does hero change through horror experience?
- [ ] **Resourceful**: Does hero show competence/skills?

### Horror Archetype

**If Final Girl/Guy:**
- [ ] Pure/moral center (or flaw makes them relatable)
- [ ] Resourceful and observant
- [ ] Survives through competence, not just luck
- [ ] Confronts threat in climax

**If Skeptic:**
- [ ] Rational worldview established
- [ ] Transformation to believer earned
- [ ] Skepticism feels reasonable (not stupidity)

**If Protector:**
- [ ] Protective bond clear
- [ ] Protection tested (impossible choices)
- [ ] Self-sacrifice considered or made

**If Expert:**
- [ ] Expertise established and believable
- [ ] Knowledge earned (not just convenient)
- [ ] Cost of knowledge shown

### Supporting Cast

- [ ] **Each Character Has Purpose**: Every character serves story function?
- [ ] **First Victim Impactful**: First death creates stakes?
- [ ] **Character Deaths Meaningful**: Deaths affect story/hero?
- [ ] **No Redundant Characters**: No characters serving same function?

---

## Phase 3: Structure Check

### Act One (Setup - 25%)

- [ ] **Normal World Established**: Do we see hero's life before horror?
- [ ] **Inciting Incident Clear**: First horror event clear and dramatic?
- [ ] **First Scare Effective**: Does it demonstrate threat?
- [ ] **Awareness Struggle**: How does hero react? Denial? Investigation?
- [ ] **Commitment**: Does hero commit to confrontation/escape by Act One end?

### Act Two (Confrontation - 50%)

**Early Encounters:**
- [ ] Character experiences threat (learns rules)
- [ ] Threat demonstrated (show, don't just tell)
- [ ] Character makes mistakes (realistic given fear)

**Midpoint:**
- [ ] Major revelation or escalation
- [ ] Stakes raised (threat worse than thought)
- [ ] Game changes (character in deeper)

**Escalation:**
- [ ] Threat becomes more active/dangerous
- [ ] Character loses ground (resources, allies, safe spaces)
- [ ] Each encounter tops previous

**All Is Lost:**
- [ ] Character's lowest point
- [ ] Seemingly no hope
- [ ] Major ally lost or character alone

**Breakthrough:**
- [ ] Character finds solution/approach
- [ ] Hope returns (but not guaranteed)

### Act Three (Resolution - 25%)

- [ ] **Final Confrontation**: Hero faces threat directly
- [ ] **Climax Intensity**: Maximum terror/horror
- [ ] **Resolution Earned**: Solution earned through character growth
- [ ] **Hero Changed**: Does ending show transformation?
- [ ] **Emotional Payoff**: Is ending satisfying?

---

## Phase 4: Scare Design Check

### Scare Variety

- [ ] **Not All Jump Scares**: Mix with dread, terror, psychological, gross-out?
- [ ] **Scare Escalation**: Each scare more intense than last?
- [ ] **Vary Scare Locations**: Different places/settings?
- [ ] **Vary Scare Times**: Not all at night?

### Tension Build

- [ ] **Tension Before Scare**: Each scare has buildup?
- [ ] **Release Satisfying**: Payoff matches buildup?
- [ ] **False Alarms Varied**: Not overused (reader becomes desensitized)?
- [ ] **Aftermath Shown**: Character processes scare?

### Sensory Details

- [ ] **Visual Scary**: Effective imagery?
- [ ] **Auditory Scary**: Sound used (silence, noise, music)?
- [ ] **Other Senses**: Smell, touch, body reactions?
- [ ] **Multi-Sensory**: More than one sense engaged?

### Pacing

- [ ] **Tension-Release Cycle**: Alternates dread/terror?
- [ ] **Not Constant Terror**: Breathing room between scares?
- [ ] **Escalation**: Pacing accelerates toward climax?
- [ ] **Rhythmic Variation**: Varied scare patterns?

---

## Phase 5: Setting and Atmosphere Check

### Setting

- [ ] **Isolation Clear**: Characters cut off from help?
- [ ] **Setting Threatening**: Does setting create danger?
- [ ] **Setting Character**: Is setting active (not passive)?
- [ ] **Atmosphere Strong**: Mood established (dread, wrongness)?
- [ ] **Sensory Details**: Multi-sensory atmosphere?

### Atmosphere

- [ ] **Dread Created**: Pervasive sense of wrongness?
- [ ] **Visual Atmosphere**: Dark, shadows, disturbing imagery?
- [ ] **Auditory Atmosphere**: Silence, sounds, music?
- [ ] **Olfactory/Tactile**: Smells, textures, temperature?
- [ ] **Psychological Atmosphere**: Character's fear becomes reader's?

---

## Phase 6: Stakes Check

- [ ] **Personal Stakes**: What hero loses if they fail?
- [ ] **Existential Stakes**: What happens to life/sanity?
- [ ] **Universal Stakes**: What happens to others/world?
- [ ] **Stakes Escalate**: Stakes rise Act One → Act Two → Act Three?
- [ ] **Stakes Matter**: Does reader care about stakes?

---

## Phase 7: Theme Check

- [ ] **Primary Theme Clear**: What is story REALLY about?
- [ ] **Horror Explores Theme**: How does threat reveal theme?
- [ ] **Thematic Depth**: Does story explore theme deeply?
- [ ] **Character Supports Theme**: Does hero's arc illustrate theme?
- [ ] **Ending Resonates**: Does ending thematically resonate?

---

## Phase 8: Consistency Check

### Internal Consistency

- [ ] **No Rule Breaking**: Do you never break established threat rules?
- [ ] **Cause and Effect**: All consequences logical based on rules?
- [ ] **Character Actions Consistent**: Character behavior believable?
- [ ] **Setting Consistent**: Setting doesn't change arbitrarily?

### Reader Trust

- [ ] **Trust Earned**: Do you maintain reader's suspension of disbelief?
- [ ] **No Convenient Fixes**: No deus ex machina?
- [ ] **No Unexplained Events**: Everything explained by rules?
- [ ] **No Contradictions**: No contradictions within story logic?

---

## Phase 9: Common Horror Problems Check

### Problem: Not Scary

- [ ] Hero not vulnerable: ☐ Fixed - Hero genuinely endangered
- [ ] No empathy: ☐ Fixed - Hero relatable, care about survival
- [ ] No isolation: ☐ Fixed - Help cannot arrive
- [ ] Rushed scares: ☐ Fixed - Build tension before each scare
- [ ] Generic setting: ☐ Fixed - Unique, atmospheric setting

### Problem: Too Many Jump Scares

- [ ] Constant startles: ☐ Fixed - Vary scare types
- [ ] No tension build: ☐ Fixed - Dread accumulation before release
- [ ] All scares same: ☐ Fixed - Mix dread, terror, psychological

### Problem: Character Stupidity

- [ ] Unrealistic decisions: ☐ Fixed - Decisions make sense in context
- [ ] Fear explains judgment: ☐ Fixed - Poor decisions due to panic
- [ ] Limited information: ☐ Fixed - Character doesn't know reader knows
- [ ] No better options: ☐ Fixed - Character trapped, no choice

### Problem: Bland Protagonist

- [ ] No personality: ☐ Fixed - Establish character before horror
- [ ] No flaws: ☐ Fixed - Character flawed but human
- [ ] No voice: ☐ Fixed - Unique perspective/dialogue
- [ ] No personal stakes: ☐ Fixed - What matters to character?

### Problem: Static Character

- [ ] No growth: ☐ Fixed - Horror transforms character
- [ ] No arc: ☐ Fixed - Character changes through experience
- [ ] No trauma response: ☐ Fixed - Character affected by horror

---

## Phase 10: Final Polish

### Prose

- [ ] **Active Voice**: Is writing in active voice?
- [ ] **Strong Verbs**: Are verbs specific and strong?
- [ ] **Show Don't Tell**: Are emotions shown through action?
- [ ] **Sensory Details**: Are sensory details included?

### Dialogue

- [ ] **Character Voices**: Does each character have distinct voice?
- [ ] **Subtext**: Is dialogue subtextual (not on-the-nose)?
- [ ] **Natural Under Pressure**: Does dialogue sound like terrified people?

### Gore and Violence (if applicable)

- [ ] **Purpose**: Does gore serve story/character?
- [ ] **Not Gratuitous**: Is gore balanced with other scare types?
- [ ] **Suggest More Than Show**: Is imagination used (often scarier)?
- [ ] **Visceral Impact**: Does gore have emotional impact?

---

## Phase 11: Reader Experience Check

### Fear Factor

- [ ] **Genuinely Frightening**: Does story elicit real fear?
- [ ] **Dread Created**: Does story create anticipation/anxiety?
- [ ] **Terror Moments**: Are there in-the-moment terror scenes?
- [ ] **Horror Elements**: Are there disturbing/revolting moments (if appropriate)?
- [ ] **Lasting Impact**: Does horror stay with reader after finishing?

### Engagement

- [ ] **Hook Strong**: Does story hook reader in first chapter?
- [ ] **Pacing Holds Attention**: Does story maintain reader interest?
- [ ] **Questions Raised**: Does story raise interesting questions?
- [ ] **Emotional Investment**: Does reader care about characters?

### Satisfaction

- [ ] **Setup Paid Off**: Are all setups paid off?
- [ ] **Questions Answered**: Are raised questions answered?
- [ ] **Ending Satisfying**: Is ending emotionally satisfying?
- [ ] **Relevant Meaning**: Does story feel meaningful?

---

## Phase 12: Subgenre-Specific Checks

### If Supernatural Horror

- [ ] **Supernatural Rules Clear**: How does entity operate?
- [ ] **Rules Followed**: Never break established rules?
- [ ] **Supernatural Scary**: Entity genuinely frightening?

### If Psychological Horror

- [ ] **Reality Ambiguous**: Is reader unsure what's real?
- [ ] **Psychology Compelling**: Madness/paranoia believable?
- [ ] **Resolution Satisfying**: Ambiguity or clarity, either works?

### If Slasher

- [ ] **Killer Scary**: Killer genuinely threatening?
- [ ] **Victim Variety**: Different victim types?
- [ ] **Final Girl Earned**: Survival through competence?
- [ ] **Innovative**: Not just rehashing Halloween/Friday the 13th?

### If Creature Feature

- [ ] **Creature Scary**: Monster genuinely frightening?
- [ ] **Creature Rules**: How does it operate/hunt?
- [ ] **Escalation**: Creature bolder/numerous?

### If Body Horror

- [ ] **Body Horror Visceral**: Transformations/infections disturbing?
- [ ] **Body as Character**: Is body horror central to story?
- [ ] **Meaning**: Does body horror explore theme?

### If Zombie Apocalypse

- [ ] **Zombie Scary**: Undead genuinely frightening?
- [ ] **Spread Clear**: How does infection spread?
- [ ] **Survival Mechanics**: How do characters survive?

## Final Checklist

### Essentials (Must Have)

- [ ] Clear threat with consistent rules
- [ ] Vulnerable, empathetic hero
- [ ] Isolated setting (no easy escape)
- [ ] High stakes (personal, existential, universal)
- [ ] Satisfying climax (earned, not accidental)
- [ ] Character growth through horror

### Strengths (Should Have)

- [ ] Compelling hook and concept
- [ ] Unique take on horror subgenre
- [ ] Emotional depth and thematic resonance
- [ ] Memorable scare scenes
- [ ] Strong supporting cast
- [ ] Atmospheric setting

### Polish (Nice to Have)

- [ ] Beautiful prose
- [ ] Clever scare sequences
- [ ] Surprise revelations (that fit setup)
- [ ] Perfect pacing (dread → terror → horror rhythm)
- [ ] Original voice
- [ ] Genre blend works seamlessly

## Pre-Publication Final Questions

1. **Is this story genuinely frightening?**
   - If no: Strengthen scares, build tension, increase stakes
   - If yes: ✓ Proceed

2. **Can reader articulate the threat and its rules?**
   - If no: Make threat clearer
   - If yes: ✓ Proceed

3. **Does hero earn their survival/defeat of threat?**
   - If no: Make resolution earned through growth/competence
   - If yes: ✓ Proceed

4. **Would story work without horror element?**
   - If no: Horror is essential - ✓ Proceed
   - If yes: Strengthen character (horror should serve character)

5. **Is every scene either character development OR scare/plot advancement (ideally both)?**
   - If no: Cut or combine scenes
   - If yes: ✓ Proceed

**If you answered yes to all five questions, your horror story is ready for publication!**

## Developmental Edit Toolkit

### Developmental Edit Checklist
- [ ] Pacing map created (fast/slow alternation)
- [ ] Tension curve visible with rises and releases
- [ ] Stakes escalate at every major turning point
- [ ] Theme statement articulated in one sentence
- [ ] Key scenes reinforce theme (no thematic drift)
- [ ] Character arcs align with theme and stakes
- [ ] Subplots either serve theme or are removed

### Chapter/Scene Review Prompts (Plot + Tension)
For each chapter/scene/section, answer:
- What changes in story state by the end?
- How does it advance or complicate the main plot?
- What new question or uncertainty does it create?
- Where does it land on the tension curve (rise, peak, release)?
- How do stakes shift (higher, more personal, more urgent)?
- What is the reader's emotional beat?
- Is there a clear hook or propulsive end?

### Restructuring Tools
- [ ] Reorder scenes to improve cause-and-effect or escalation
- [ ] Merge adjacent scenes that repeat information
- [ ] Split overloaded scenes into distinct beats
- [ ] Remove scenes that do not change story state
- [ ] Replace exposition-only scenes with active conflict
- [ ] Insert bridging beats to smooth gaps after cuts

### Approval Gate: Before Line Edit
Proceed to line-level editing only when:
- [ ] Structural issues resolved (plot holes, order, missing beats)
- [ ] Pacing and tension curve reviewed and adjusted
- [ ] Stakes escalation verified across the story
- [ ] Theme tracker filled and alignment confirmed
- [ ] Restructuring actions completed
- [ ] Feedback incorporated (beta/editor notes, if available)
