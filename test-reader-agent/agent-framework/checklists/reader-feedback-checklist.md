# Reader Feedback Checklist

Comprehensive checklist for providing complete reader feedback.

---

## PRE-READING PREPARATION

### Understanding Context

- [ ] Target audience identified
- [ ] Genre/subgenre confirmed
- [ ] Comparable titles known (if provided)
- [ ] Special requirements or questions noted
- [ ] Length/format understood

### Reading Setup

- [ ] Tracking method ready (markers, notes)
- [ ] Worksheet prepared
- [ ] Reading in realistic conditions
- [ ] Reader mindset (not editor/writer)
- [ ] Fresh perspective (not pre-judging)

---

## DURING READING

### Engagement Tracking

- [ ] Marking engagement level per chapter
- [ ] Noting high-engagement moments
- [ ] Noting low-engagement moments
- [ ] Tracking where I want to stop
- [ ] Tracking where I can't stop

### Emotional Tracking

- [ ] Marking emotional responses
- [ ] Noting strong emotions (positive)
- [ ] Noting strong emotions (negative)
- [ ] Noting emotional flatness
- [ ] Noting failed emotional beats

### Confusion Tracking

- [ ] Marking confusion points immediately
- [ ] Noting what specifically confused
- [ ] Tracking if confusion resolves
- [ ] Noting impact of confusion

### Believability Tracking

- [ ] Marking moments of disbelief
- [ ] Noting what didn't work
- [ ] Tracking impact on immersion

### Quick Notes

- [ ] Brief notes on notable moments
- [ ] Not over-analyzing while reading
- [ ] Marking for later rather than stopping
- [ ] Maintaining reader flow

---

## IMMEDIATELY AFTER READING

### First Impressions

- [ ] Captured gut reaction
- [ ] Noted what I remember most
- [ ] Recorded immediate emotional state
- [ ] Wrote one-sentence summary
- [ ] Assigned initial ratings

### Quick Assessment

- [ ] Overall impression captured
- [ ] Best moments identified
- [ ] Worst moments identified
- [ ] Satisfaction level recorded
- [ ] Would-recommend decision made

---

## DETAILED ANALYSIS

### Engagement Analysis

- [ ] Created engagement map
- [ ] Identified patterns in engagement
- [ ] Analyzed high-engagement sections
- [ ] Analyzed low-engagement sections
- [ ] Assessed overall engagement arc

### Emotional Analysis

- [ ] Reviewed emotional marks
- [ ] Identified emotional journey
- [ ] Assessed character investment
- [ ] Identified earned emotional moments
- [ ] Identified unearned emotional moments
- [ ] Evaluated emotional satisfaction

### Confusion Analysis

- [ ] Reviewed all confusion points
- [ ] Categorized by type
- [ ] Assessed severity
- [ ] Noted resolution status
- [ ] Evaluated overall clarity

### Pacing Analysis

- [ ] Identified slow sections
- [ ] Identified fast sections
- [ ] Assessed overall pacing
- [ ] Compared to genre expectations
- [ ] Noted pacing patterns

### Genre Analysis

- [ ] Evaluated core genre experience
- [ ] Checked required elements
- [ ] Assessed convention compliance
- [ ] Evaluated subversions (if any)
- [ ] Assessed genre ending satisfaction

### Believability Analysis

- [ ] Reviewed all believability breaks
- [ ] Assessed severity and impact
- [ ] Noted patterns
- [ ] Evaluated overall suspension of disbelief

---

## PATTERN IDENTIFICATION

### Finding Patterns

- [ ] Identified recurring issues
- [ ] Identified recurring strengths
- [ ] Grouped related issues
- [ ] Assessed cumulative impact
- [ ] Distinguished patterns from one-offs

---

## PRIORITIZATION

### Ranking Issues

- [ ] Separated critical from minor issues
- [ ] Ordered concerns by impact
- [ ] Identified top priorities
- [ ] Distinguished personal from universal
- [ ] Confirmed prioritization makes sense

---

## REPORT PREPARATION

### Organizing Feedback

- [ ] Overall impression written
- [ ] Engagement map included
- [ ] Confusion points documented
- [ ] Emotional response covered
- [ ] Pacing assessment included
- [ ] Genre expectations addressed
- [ ] Strengths documented
- [ ] Concerns documented and prioritized
- [ ] Target audience fit assessed

### Feedback Quality Check

- [ ] All feedback is specific (location + effect)
- [ ] Feedback is descriptive, not prescriptive
- [ ] Intensity is indicated
- [ ] Personal vs. universal distinguished
- [ ] Tone is constructive
- [ ] Positives included with negatives

### Format Check

- [ ] Using appropriate template
- [ ] All sections completed
- [ ] Clear organization
- [ ] Easy to navigate
- [ ] Professional presentation

---

## SPECIFIC ELEMENT CHECKS

### Opening Assessment

- [ ] Hook effectiveness evaluated
- [ ] Orientation clarity assessed
- [ ] Investment building tracked
- [ ] First chapter engagement noted
- [ ] Opening promises identified

### Middle Assessment

- [ ] Momentum maintenance evaluated
- [ ] Saggy sections identified
- [ ] Conflict escalation tracked
- [ ] Character development noted
- [ ] Reader investment at midpoint assessed

### Ending Assessment

- [ ] Climax impact evaluated
- [ ] Resolution satisfaction assessed
- [ ] Promise fulfillment checked
- [ ] Emotional payoff rated
- [ ] Final impression captured

### Character Assessment

- [ ] Protagonist investment tracked
- [ ] Investment timing noted
- [ ] Secondary character impact noted
- [ ] Relationship investment tracked
- [ ] Character believability assessed

---

## PERSONAL BIAS CHECK

### Self-Awareness

- [ ] Noted personal preferences that may affect feedback
- [ ] Distinguished personal taste from craft issues
- [ ] Considered target audience perspective
- [ ] Flagged potentially personal reactions
- [ ] Adjusted feedback where appropriate

---

## FINAL REPORT CHECK

### Completeness

- [ ] All required sections filled
- [ ] All significant issues covered
- [ ] All significant strengths noted
- [ ] Patterns identified
- [ ] Priorities clear

### Usefulness

- [ ] Feedback is actionable (even if descriptive)
- [ ] Locations are specific
- [ ] Effects are clear
- [ ] Patterns are highlighted
- [ ] Priorities guide attention

### Constructiveness

- [ ] Tone is respectful
- [ ] Feedback serves improvement
- [ ] Strengths balanced with concerns
- [ ] Harsh truths delivered kindly
- [ ] Overall impression fair

### Accuracy

- [ ] Feedback represents true experience
- [ ] No exaggeration or softening
- [ ] Personal bias acknowledged
- [ ] Audience perspective maintained
- [ ] Genre context considered

---

## DELIVERY PREPARATION

### Before Sending

- [ ] Proofread report
- [ ] Verified all references correct
- [ ] Confirmed nothing missing
- [ ] Ensured professional presentation
- [ ] Ready to answer questions
