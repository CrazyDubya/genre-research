# Murder Mystery Revision Checklist

## Structure and Plot

### Overall Structure
- [ ] Three-act structure clear and functional
- [ ] Pacing varies appropriately
- [ ] Transitions between scenes smooth
- [ ] Chapter breaks create hooks
- [ ] No dragging sections
- [ ] No rushed sections

### The Setup
- [ ] Hook works and grabs attention
- [ ] Crime committed early enough
- [ ] Detective introduced properly
- [ ] World established clearly
- [ ] Stakes established
- [ ] Central questions clear

### The Investigation
- [ ] Each suspect adequately investigated
- [ ] Clues discovered at logical pace
- [ ] Red herrings compelling but false
- [ ] Complications arise naturally
- [ ] Midpoint changes direction
- [ ] Stakes raised appropriately

### The Resolution
- [ ] Solution feels surprising but inevitable
- [ ] All clues explained
- [ ] All red herrings explained
- [ ] Killer identification earned
- [ ] Motive clear and compelling
- [ ] Method detailed and believable
- [ ] Loose ends tied up
- [ ] Subplots resolved

## Characters

### Protagonist
- [ ] Voice consistent and distinct
- [ ] Skills and limitations established
- [ ] Character arc complete
- [ ] Flaws and weaknesses present
- [ ] Personal stakes clear
- [ ] Methods explained
- [ ] Growth demonstrated

### Supporting Cast
- [ ] Each character distinct
- [ ] Each serves purpose
- [ ] No cardboard characters
- [ ] Voices distinct from each other
- [ ] Relationships clear
- [ ] Secrets revealed appropriately

### Killer
- [ ] Introduced early
- [ ] Clues point to them (however subtly)
- [ ] Motive established
- [ ] Means and opportunity shown
- [ ] Not too obvious
- [ ] Not impossible to guess

### Victim
- [ ] Fully realized character
- [ ] Not just a plot device
- [ ] Relationships clear
- [ ] Secrets revealed
- [ ] Death means something

## Clues and Evidence

### Essential Clues
- [ ] All essential clues present
- [ ] Planted early enough
- [ ] Discoverable by reader
- [ ] Explained in resolution
- [ ] Lead to solution logically

### Clue Distribution
- [ ] Clues distributed throughout
- [ ] Not all revealed at once
- [ ] Not all held until end
- [ ] Some recontextualized
- [ ] Some immediately obvious
- [ ] Some require thought

### Red Herrings
- [ ] Each red herring serves additional purpose
- [ ] Each explained in resolution
- [ ] Not too obvious
- [ ] Not too subtle
- [ ] Compelling alternatives
- [ ] Fairly dismissed

### Fair Play
- [ ] No withheld information
- [ ] No last-minute evidence
- [ ] No supernatural (unless genre)
- [ ] No coincidences
- [ ] Reader has same info as detective
- [ ] Solution traceable from clues

## Dialogue

- [ ] Information delivered naturally

### Interviews and Interrogations

- [ ] Power dynamics shift
- [ ] Tension present
- [ ] Information controlled

### Clues in Dialogue
- [ ] Clues embedded naturally
- [ ] Not too obvious
- [ ] Not easily missed
- [ ] Can be found on reread
- [ ] Recontextualizable

## Beta Reader Feedback

### Questions to Ask:
1. Who did you suspect? When?
2. Did you figure it out? When?
3. Were there any boring parts?
4. Any parts confusing?
5. Did solution feel earned?
6. Did clues feel fair?
7. What was your favorite moment?
8. What was your least favorite?
9. Would you read more by this author?
10. Any specific suggestions?

### Common Issues to Watch:
- "I figured it out immediately" → too obvious
- "I never could have guessed that" → not fair
- "I got bored in the middle" → pacing issue
- "I didn't understand why X happened" → logic issue
- "The ending felt rushed" → pacing issue

## Readability, Vocabulary, and Style Checks
- [ ] Readability score measured (e.g., Flesch-Kincaid) matches target audience.
- [ ] Sentence length distribution supports intended pace.
- [ ] Vocabulary level consistent (no random shifts in formality/complexity).
- [ ] Jargon/slang is either clear from context or defined.
- [ ] Repetitive words/phrases reduced.

## Pacing Analysis (Scene Length + Tension Curve)
- [ ] Scene lengths tracked (word/page count) and vary intentionally.
- [ ] Long scenes broken with beats or transitions where needed.
- [ ] Tension curve plotted across the story (rises, releases, peaks).
- [ ] Major releases placed after high-tension scenes.

## Tone and Voice Consistency Audit
- [ ] Narrative tone consistent with genre expectations.
- [ ] POV voice stable across chapters/sections.
- [ ] Character voices maintain consistent diction and cadence.
- [ ] Tonal shifts are deliberate and signposted.

## Target Audience Alignment
- [ ] Target audience defined (age, subgenre, sensitivity).
- [ ] Content intensity matches audience expectations.
- [ ] Complexity matches audience comfort (plot density, jargon, mechanics).
- [ ] Hook/logline reflects what the audience wants.
