# Writer Self-Check

Use this checklist to evaluate your own work before submission.

---

## PRE-SUBMISSION CHECKLIST

### Before Submitting Any Milestone

- [ ] Required word count met (within acceptable range)
- [ ] All required scenes/chapters included
- [ ] Format specifications followed
- [ ] File named correctly
- [ ] Previous feedback addressed
- [ ] Self-revision pass completed
- [ ] Read-through completed

---

## STORY-LEVEL CHECK

### Structure

- [ ] Act One establishes setup and inciting incident
- [ ] Act Two maintains momentum and escalates conflict
- [ ] Act Three delivers climax and resolution
- [ ] Midpoint shift is present and effective
- [ ] Pacing feels appropriate throughout

### Plot

- [ ] Central conflict is clear and sustained
- [ ] Cause-and-effect logic holds
- [ ] No major plot holes
- [ ] All significant plot threads addressed (for current milestone)
- [ ] Stakes are clear and escalating
- [ ] Ending (if present) feels earned

### Character

- [ ] Protagonist is compelling and active
- [ ] Character goal is clear
- [ ] Character arc is progressing
- [ ] Motivations are understandable
- [ ] Character consistency maintained
- [ ] Supporting characters serve their purpose
- [ ] Antagonist/opposition is effective

---

## SCENE-LEVEL CHECK

For each significant scene:

### Purpose and Function

- [ ] Scene has clear purpose (advances plot/character/information)
- [ ] Something changes by scene's end
- [ ] Scene earns its place in the story

### Scene Structure

- [ ] Scene has conflict or tension
- [ ] POV character has goal
- [ ] Scene enters late and exits early
- [ ] Scene connects to surrounding scenes

### Scene Elements

- [ ] Appropriate balance of dialogue/action/description/interiority
- [ ] Grounded in setting
- [ ] Pacing appropriate to content

---

## PROSE-LEVEL CHECK

### Clarity

- [ ] Meaning is clear on first read
- [ ] No confusing passages
- [ ] Reader can follow action/sequence
- [ ] Transitions work

### Voice

- [ ] Voice is consistent
- [ ] POV is maintained
- [ ] Character voice is appropriate
- [ ] Tone matches content

### Quality

- [ ] Sentences are varied in length and structure
- [ ] Word choice is precise
- [ ] Unnecessary words are cut
- [ ] Rhythm serves content

---

## DIALOGUE CHECK

- [ ] Dialogue sounds natural when read aloud
- [ ] Characters sound distinct from each other
- [ ] Dialogue advances story or reveals character
- [ ] Subtext is present where appropriate
- [ ] Tags and beats are effective
- [ ] No floating/talking heads sections
